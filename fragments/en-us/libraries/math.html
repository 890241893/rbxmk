Extensions to the standard math library.

The math library is an extension to the standard library that includes the same additions to Roblox's math library:

$FIELDS

Returns a number clamped between a minimum and maximum.

The clamp function returns x clamped so that it is not less than min or greater than max.

Includes optional base argument.

The log function returns the logarithm of x in base. The default for base is e, returning the natural logarithm of x.

Rounds a number to the nearest integer.

The round function returns x rounded to the nearest integer. The function rounds half away from zero.

Returns the sign of a number.

The sign function returns the sign of x: 1 if x is greater than zero, -1 of x is less than zero, and 0 if x equals zero.
