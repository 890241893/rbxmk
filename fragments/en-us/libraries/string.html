Extensions to the standard string library.

The string library is an extension to the standard library that includes the same additions to Roblox's string library.

Splits a string into a list of substrings.

The split function splits s into substrings separated by sep.

If sep is nil, or if sep is not nil but not in s, then split returns a table with s as its only element.

If sep is empty, then s is split after each UTF-8 sequence.

Note: Roblox's implementation splits per byte, while this implementation splits per UTF-8 character.
