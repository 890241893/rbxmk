Operations for handling JSON data.

The json library provides functions for handling JSON data.

Get difference between two JSON values.

The diff function returns a patch that represents the transformation of prev to next. This patch can be applied with json.patch.

Converts a string to a JSON value.

The fromString function attempts to convert string to a JSON value. It is equivalent to rbxmk.decodeFormat("json", string)

Tranforms a JSON value according to a patch.

The patch function returns a copy of value that has been transformed according to patch.
