Display the version.

The version command displays the current version of rbxmk. The result is formatted according to semantic versioning.

Format of the version output. Available formats are "text", "json".

Increase verbosity level by 1 for each time the flag is specified. Default is level 0. Each level appends the following information:

Level Description
0 Includes program version.
1 Includes basic build information.
2 Includes detailed build information.
3 Includes module information.
