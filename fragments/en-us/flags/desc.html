The --desc-* flags set the rbxmk.globalDesc field. They also set the Enum global variable to the enums generated from the descriptor.

Include the file located at `path` as a global descriptor. May be specified any number of times. Multiple descriptors are merged in the order specified.

Download and start with the latest API dump as the global descriptor.

Include the file located at `path` as a patch to the global descriptor. May be specified any number of times. Multiple patches are applied in the order specified.
