Encodes binary model data.

The rbxm format encodes Instances in the Roblox binary model format.

Direction Type Description
Decode DataModel A DataModel instance.
Encode DataModel A DataModel instance.
Encode Instance A single instance, interpreted as a child to a DataModel.
Encode Objects A list of Instances, interpreted as children to a DataModel.

This format has the same options as the rbxl format.
