A two-dimensional Euclidean vector with 16-bit integer precision.

The Vector2int16 type represents a two-dimensional Euclidean vector with 16-bit integer precision.

Returns the zero vector.

The new constructor returns the origin vector, where each coordinate is 0.

Returns a vector composed by each given coordinate.

The new constructor returns a vector composed by each given component.

The X coordinate.

The X field returns the X coordinate of the vector.

The Y coordinate.

The Y field returns the Y coordinate of the vector.

The sum of two vectors.

The add operator returns a vector where each corresponding component of the two operands are summed.

The difference between two vectors.

The sub operator returns a vector where each corresponding component of the two operands are subtracted.

Multiplies each corresponding component.

The mul operator returns a vector where each corresponding component of the two operands are multiplied.

Multiplies each component.

The mul operator returns a vector where each component of the first operand is multiplied by the second operand.

Divides each corresponding component.

The div operator returns a vector where each corresponding component of the two operands are divided.

Divides each component.

The div operator returns a vector where each component of the first operand is divided by the second operand.

Returns whether two Vector2int16 values are equal.

The equal operator returns true if both operands are Vector2int16, and each corresponding component is equal.

The negation of the vector.

The unm operator returns a vector where each component is negated.
