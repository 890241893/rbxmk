An axis-aligned rectangle.

The Rect type represents an axis-aligned two-dimensional rectangle with a lower and upper boundary.

Returns a Rect composed by two Vector2s.

The new constructor returns a Rect composed by two Vector2s. min is the lower bounds, and max is the upper bounds.

Returns a Rect composed by boundary components. minX and minY compose the lower bounds, and maxX and maxY compose the upper bounds.

The new constructor returns a Rect composed by boundary components.

The height of the rectangle.

The Height field returns the height of the rectangle, or the difference between the upper and lower bounds on the Y axis.

The upper bounds of the rectangle.

The Max field returns the upper bounds of the rectangle.

The lower bounds of the rectangle.

The Min field returns the lower bounds of the rectangle.

The width of the rectangle.

The Width field returns the width of the rectangle, or the difference between the upper and lower bounds on the X axis.

Returns whether two Rect values are equal.

The equal operator returns true if both operands are Rect, and each corresponding component is equal.
