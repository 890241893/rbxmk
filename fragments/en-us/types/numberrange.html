A range of numbers.

The NumberRange type represents a range of numbers between a minimum and maximum.

Returns a range with a single value.

The new constructor returns a NumberRange where value is both the minimum and maximum.

Returns a range with between two values.

The new constructor returns a NumberRange where minimum is the minimum value and maximum is the maximum value. Throws an error if minimum is greater than maximum.

The maximum value.

The Max field returns the maximum possible value in the range.

The minimum value.

The Min field returns the minimum possible value in the range.

Returns whether two NumberRange values are equal.

The equal operator returns true if both operands are NumberRange, and each corresponding component is equal.
