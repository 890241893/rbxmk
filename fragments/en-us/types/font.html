The font used to render text.

The Font type represents the description of a font.

Creates a new Font.

The new constructor returns a Font composed of the components from each argument.

A sequential value.

The CachedFaceId property is an internal value representing a path to resolved font data.

Path to a font family.

The Family property points to a font file.

The weight of font text.

The Weight property determines the thickness of the text. The value is a standard OpenType weight value, where 400 is regular, 700 is bold, 100 is thin, and 900 is heavy.

Whether the text is italic.

The Style property determines whether the text is italic. 0 means normal, and 1 means italic.

Returns whether two Font values are equal.

The equal operator returns true if both operands are Font, and each corresponding component is equal.
