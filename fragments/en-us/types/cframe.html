A position and rotation.

The CFrame type is a matrix representing a combined position and orientation.

Returns a CFrame composed from angles.

The Angles constructor returns a CFrame located at the origin, oriented according to the angles (rx, ry, rz), in radians. Rotations are ordered Z, Y, X.

Returns a CFrame composed from an axis and angle.

The fromAxisAngle constructor returns a CFrame located at the origin, rotated rotation radians around axis.

Returns a CFrame composed from Euler angles.

The fromEulerAnglesXYZ constructor returns a CFrame located at the origin, oriented according to the angles (rx, ry, rz), in radians. Rotations are ordered Z, Y, X.

Returns a CFrame composed from Euler angles ordered Y, X, Z.

The fromEulerAnglesYXZ constructor returns a CFrame located at the origin, oriented according to the angles (rx, ry, rz), in radians. Rotations are ordered Y, X, Z.

Returns a CFrame from a matrix.

The fromMatrix constructor returns a CFrame located at position, rotated according to the following rotation matrix:

[vx.X, vy.X, vz.X]
[vx.Y, vy.Y, vz.Y]
[vx.Z, vy.Z, vz.Z]

If vz is the zero vector, then vz is calculated as the unit of the cross product of vx and vy.

Returns a CFrame composed from orientations.

The fromOrientation constructor returns a CFrame located at the origin, oriented according to the angles (rx, ry, rz), in radians. Rotations are ordered Y, X, Z.

Returns a CFrame that points towards a location.

The lookAt constructor returns a CFrame located at position, facing towards lookAt, with the local upward direction determined by up.

Returns the default CFrame.

The new constructor returns a CFrame with the default orientation located at the origin.

Returns a CFrame with a position.

The new constructor returns a CFrame with the default orientation located at position.

Returns a CFrame pointing towards a location while tilted upward.

The new constructor returns a CFrame located at position, facing towards lookAt. The local upward direction is (0, 1, 0).

Returns a CFrame with a position from coordinates.

The new constructor returns a CFrame located at (x, y, z), with the default orientation.

Returns a CFrame from a quaternion.

The new constructor returns a CFrame located at (x, y, z), oriented according to the quaternion (qx, qy, qz, qw).

Returns a CFrame from raw components.

The new constructor returns a CFrame located at (x, y, z), oriented according to the following rotation matrix:

[r00, r01, r02]
[r10, r11, r12]
[r20, r21, r22]

The forward direction.

The LookVector field returns the forward-direction, or the negation of the third column of the rotation matrix.

The position of the CFrame.

The P field returns the position of the CFrame.

The position of the CFrame.

The Position field returns the position of the CFrame.

The right direction.

The RightVector field returns the right-direction, or first column of the rotation matrix.

The up direction.

The UpVector field returns the up-direction, or second column of the rotation matrix.

The X component of the Position.

The X field returns the X component of the Position.

The first row of the rotation matrix.

The XVector field returns the first row of the rotation matrix.

The Y component of the Position.

The Y field returns the Y component of the Position.

The second row of the rotation matrix.

The YVector field returns the second row of the rotation matrix.

The Z component of the Position.

The Z field returns the Z component of the Position.

The third row of the rotation matrix.

The ZVector field returns the third row of the rotation matrix.

Returns the raw components.

The GetComponents method returns the components of the CFrame's position and rotation matrix.

Returns the inverse CFrame.

The Inverse method returns the inverse of the CFrame.

Linearly interpolates between two CFrames.

The Lerp method returns a CFrame linearly interpolated from the CFrame to goal according to alpha, which has an interval of [0, 1].

Transforms a Vector3 to local space.

The PointToObjectSpace method returns a Vector3 transformed from world to local space of the CFrame.

Transforms a Vector3 to world space.

The PointToWorldSpace method returns a Vector3 transformed from local to world space of the CFrame.

Returns the orientation as an axis and angle.

The ToAxisAngle method returns the orientation of the CFrame as an angle, in radians, rotated around an axis.

Returns the orientation as Euler angles.

The ToEulerAnglesXYZ method returns the approximate angles of the CFrame's orientation, in radians, if ordered Z, Y, X.

Returns the orientation as Euler angles order Y, X, Z.

The ToEulerAnglesYXZ method returns the approximate angles of the CFrame's orientation, in radians, if ordered Y, X, Z.

Transforms to local space.

The ToObjectSpace method returns the CFrame transformed from world to local space of cf.

Returns the orientation.

The ToOrientation method returns the approximate angles of the CFrame's orientation, in radians, if ordered Y, X, Z.

Transforms to world space.

The ToWorldSpace method returns the CFrame transformed from local to world space of cf.

Returns a Vector3 rotated to local space.

The VectorToObjectSpace method returns a Vector3 rotated from world to local space of the CFrame.

Returns a Vector3 rotated to world space.

The VectorToWorldSpace method returns a Vector3 rotated from local to world space of the CFrame.

Translates by adding.

The add operator returns the CFrame translated in world space by the operand.

Translates by subtracting.

The sub operator returns the CFrame translated in world space by the negation of the operand.

Composes two CFrames.

The mul operator returns the composition of two CFrames.

Transforms to world space.

The mul operator returns the operand transformed from local to world space of the CFrame.

Returns whether two CFrame values are equal.

The equal operator returns true if both operands are CFrame, and each corresponding component is equal.
