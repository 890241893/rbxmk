A dimension with a dynamic and constant component.

The UDim type represents one dimension with a dynamic and constant component.

Returns a new UDim.

The new constructor returns a new UDim. scale sets the Scale component, and offset sets the Offset component.

The constant component.

The Offset field returns the constant component of the UDim.

The dynamic component.

The Scale field returns the dynamic component of the UDim.

The sum of two UDim values.

The add operator returns a UDim where each corresponding component of the two operands are summed.

The difference between two UDim values.

The sub operator returns a UDim where each corresponding component of the two operands are subtracted.

Returns whether two UDim values are equal.

The equal operator returns true if both operands are UDim, and each corresponding component is equal.

The negation of the UDim.

The unm operator returns a UDim where each component is negated.
