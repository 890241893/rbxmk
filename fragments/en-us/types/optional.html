An optional exprim of a type.

The Optional type is an exprim that encapsulates another type, such that the value is either present or not present.

An Optional with no value.

The none constructor returns an empty Optional exprim of the given type.

model.WorldPivotData = Optional.none("CFrame") -- type is Optional
print(typeof(model.WorldPivotData.Value)) --> nil

An Optional with some value.

The some constructor returns an Optional exprim with the type of value, that encapsulates value.

local value = CFrame.new(1,2,3)
model.WorldPivotData = Optional.some(value) -- type is Optional
print(typeof(model.WorldPivotData.Value)) --> CFrame
