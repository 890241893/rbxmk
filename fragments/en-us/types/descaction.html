An action that transforms a descriptor.

The DescAction type describes a single action that transforms a descriptor. It has the following members:

$MEMBERS

A DescAction can be created with the [DescAction.new][DescAction.new] constructor. It is also returned from the [RootDesc.Diff][RootDesc.Diff] method and the desc-patch.json format.

Creates a new DescAction.

The new constructor returns a new DescAction initialized with a type and an element.

The type of element.

The Element property is the type of element to which the action applies.

The name of the primary element.

The Primary property is the name of the primary element. For example, the class name or enum name.

The name of the secondary element.

The Secondary property is the name of the secondary element. For example, the name of a class member or enum item. An empty string indicates that the action applies to the primary element.

The type of transformation.

The Type property is the type of transformation performed by the action.

Gets a field.

The Field method returns the value of the name field, or nil if the action has no such field.

Gets all fields.

The Fields method returns a collection of field names mapped to values.

Sets a field.

The SetField method sets the value of the name field to value.

Sets all fields.

The SetFields method replaces all fields of the action with fields.
