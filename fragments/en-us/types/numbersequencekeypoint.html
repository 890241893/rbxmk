A keypoint of a NumberSequence.

The NumberSequenceKeypoint type represents a single keypoint of a NumberSequence.

Creates a new keypoint.

The new constructor returns a new NumberSequenceKeypoint, time sets the Time field, and value sets the Value field.

Creates a new keypoint with an envelope.

The new constructor returns a new NumberSequenceKeypoint, time sets the Time field, value sets the Value field, and envelope sets the Envelope field.

The amount of variance allowed from the Value.

The Envelope field returns the amount of variance allowed from the value.

The time location along the sequence.

The Time field returns the temporal location of the keypoint in the sequence. Has an interval of [0, 1].

The base value.

The Value field returns the base value of the keypoint.

Returns whether two NumberSequenceKeypoint values are equal.

The equal operator returns true if both operands are NumberSequenceKeypoint, and each corresponding component is equal.
