A line extending infinitely in one direction.

The Ray type represents a line that extends infinitely in one direction.

Creates a new Ray.

The new constructor returns a Ray, where origin sets the Origin, and direction sets the Direction.

The direction of the ray.

The Direction field returns the direction of the ray.

The position of the ray.

The Origin field returns the position of the ray.

Returns the nearest point on the ray.

The ClosestPoint method returns the position on the ray that is nearest to point.

Returns distance to the nearest point on the ray.

The Distance method returns the distance between point and the point on the ray nearest to point.

Returns whether two Ray values are equal.

The equal operator returns true if both operands are Ray, and each corresponding component is equal.
