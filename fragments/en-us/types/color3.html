A color in RGB space.

The Color3 type represents a color in RGB space.

Returns a Color3 from hue, saturation, and value.

The fromHSV constructor returns a Color3 from the given hue, saturation, and value.

Returns a Color3 from 8-bit components.

The fromRGB constructor returns a Color3 from the given red, green, and blue components, each having an interval of [0, 255].

Returns the zero Color3.

The new constructor returns the zero value for Color3, with each component being zero, resulting in the color black.

Returns a Color3 from components.

The new constructor returns a Color3 from the given red, green, and blue components, each having an interval of [0, 1].

The blue component.

The B field returns the blue component of the color.

The green component.

The G field returns the green component of the color.

The red component.

The R field returns the red component of the color.

Linearly interpolates between two colors.

The Lerp method returns a Color3 linearly interpolated from the color to goal according to alpha, which has an interval of [0, 1].

Converts to hue, saturation, and value.

The ToHSV method returns the hue, saturation, and value approximated from the color.

Returns whether two Color3 values are equal.

The equal operator returns true if both operands are Color3, and each corresponding component is equal.
