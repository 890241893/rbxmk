Configures instance attributes.

The AttrConfig type configures how an instance encodes and decodes attributes.

$MEMBERS

Creates a new AttrConfig.

The new constructor creates a new AttrConfig. property sets the [Property][AttrConfig.Property] field, defaulting to an empty string.

The property that attributes are applied to.

The Property property determines which property of an [Instance][Instance] attributes are applied to. If an empty string, instances will default to "AttributesSerialize".
