This document contains a reference to the libraries available to rbxmk scripts.

$TOC

Lua scripts are used to perform actions in rbxmk. The environment provided by rbxmk is packaged as a set of libraries. Some libraries are loaded under a specific name, while others are loaded directly into the global environment:

$LIBRARIES

The _RBXMK_VERSION global variable is defined as a string containing the current version of rbxmk, formatted according to semantic versioning.
